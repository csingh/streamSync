Setup heroku so you can deploy code.

1. download heroku toolbelt for mac os x.
	link: https://devcenter.heroku.com/articles/getting-started-with-nodejs#set-up

2. cd to repo in terminal

3. Login kevin's heroku account
	"heroku login"

4. credentials:
	ask me

5. Add heroku git remote to repo
	"git remote add heroku https://git.heroku.com/whispering-journey-4483.git"

6. Deploy code to heroku
	"git push heroku master"

7. Open heroku site
	"heroku open"
